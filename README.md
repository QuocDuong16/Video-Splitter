# Video-Splitter
## Mô tả supermini project
*Đây là một đoạn code đơn giản để dùng cho mục đích cá nhân là chia video thành nhiều phần*

## Cài Đặt
**Hướng dẫn sử dụng:**
- Clone repo này về
- Sau đó active venv bằng lệnh
  <pre>
  .\venv\Scripts\activate
  </pre>
- Dùng lệnh này để chạy ứng dụng
  <pre>
  python .\app.py
  </pre>
